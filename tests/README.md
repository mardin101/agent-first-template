# Test Directory

This directory contains test files for your project.

## Running Tests

Add instructions for running tests:

```bash
# Example test commands
# npm test
# pytest
# go test
```

## Test Structure

Organize tests to mirror your source code structure:

```
tests/
├── unit/           # Unit tests
├── integration/    # Integration tests
└── e2e/           # End-to-end tests
```

## Writing Tests

- Follow the testing conventions in CONTRIBUTING.md
- Use the test-writer agent for assistance
- Ensure good test coverage
- Write clear, descriptive test names

## Test Coverage

Document your test coverage goals and current status.
